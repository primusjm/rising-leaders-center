# Strategy
To address Microvision's needs, we designed an interactive dashboard that effectively bridges service suppliers, buyers, and financiers. Our approach involved researching stakeholder needs, creating wireframes and prototypes, integrating real-time communication and transaction tracking features, and conducting user testing to refine the interface and improve usability.

This approach ensured a user-centric design that met the diverse requirements of all parties involved.

# User Onboarding
We focused on a seamless login and onboarding experience. The system efficiently guides users through registration, ensuring clarity and ease. Overall, our goal was to provide an intuitive, quick experience that left users eager to explore the system's features.

# Company Overview
The Company Overview section of the supplier dashboard allows users to view and edit company information, update profiles, contact details, and business status, upload documents, and track changes.

Users can also create, edit, and review deals of interest, with options to save or cancel updates to keep information accurate.

{% include media.html file="microvision/microvision-02.webp" caption="Deals & Company Overview "%}

# Milestones & Wallet
The Milestones section provides an overview of key project milestones and achievements. Users can track progress, view deadlines, and update statuses for each milestone, ensuring clear visibility and timely completion of project goals.

The User Wallet page allows users to manage financial transactions, view balances, and track payment history. It includes options to add funds, make payments, and review transaction details, offering a streamlined way to handle financial activities within the dashboard.

{% include media.html file="microvision/microvision-03.webp" caption="Milestones & User Wallet "%}

# User Dashboard & Transaction History
The Transaction History page provides a detailed record of all financial transactions. Users can view and search through past transactions, including dates, amounts, and statuses, enabling them to track financial activity and manage their accounts effectively.

The User Dashboard offers a central hub for managing personal and business activities. It displays an overview of recent activities, notifications, and key metrics, allowing users to easily navigate and access various sections of the platform.

{% include media.html file="microvision/microvision-04.webp" caption="Transaction History & User Dashboard "%}

# Outcome
The successful development of this system produced a user-friendly platform that boosts connectivity and efficiency for suppliers, buyers, and financiers. It enhances transparency and collaboration by offering detailed views, milestone tracking, transaction management, and deal reviews, leading to better decision-making and stronger business relationships.
