# Strategy
For the foundation's new website design, we implemented a sleek and user-friendly interface that prioritised ease-of-use and engagement. Utilising modern design principles, we ensured seamless navigation and intuitive functionality, allowing visitors to easily access information about the foundation's initiatives and programmes.

{% include media.html file="iaf/homepage.webp" caption="IAF Homepage"%}

By incorporating responsive design elements, the website adapts seamlessly to various devices, ensuring a consistent user experience across desktops, tablets, and mobile phones.

{% include media.html file="iaf/mobile.webp" caption="Mobile UI"%}

Additionally, we optimised the website for search engines, maximising visibility and attracting a wider audience to the foundation's cause. Through thorough testing and feedback iterations, we fine-tuned the design to align with the foundation's goals and objectives, resulting in a successful solution that effectively communicates its mission to the world.

{% include media.html file="iaf/about.webp" caption="About Us"%}

# Outcome
The new Inuka Angaza Foundation website was a success, featuring a user-friendly design and responsive layout. Key elements included an interactive donation portal, events calendar, and multimedia resource hub. The site enhanced accessibility and received positive feedback for its intuitive navigation and engaging design.
